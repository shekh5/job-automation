# Onboarding Questions

Use these only for a first-time personal DM.

1. Name or display name.
2. Target roles.
3. Experience level.
4. Location and remote preference.
5. Preferred company categories.
6. One-time help or daily cron updates.
7. Target company list, if any.
8. Resume, portfolio, GitHub, or LinkedIn link, only if the user wants to share it.
