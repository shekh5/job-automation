The Problem

Finding high-quality early-career software/tech roles is manual, rate-limited, and context-polluted. Combining multiple searches into one monolithic run triggers high LLM token costs and frequent rate-limiting failures.

  • Monolithic Failures: Scanners that crawl all sites at once hit timeout limits and Cloudflare challenges.
  • Context Pollution: Merging AI companies, Web3, services, and big-tech watchlists creates noisy, untargeted shortlists.
  • Deduplication Drag: Incrementally scanning without local state checkpoints leads to repeat recommendations.
The Multi-Agent Pipeline

We split the project into **two collaborative agents** and **category-specific slices** executing on staggered schedules:

  • Agent `main` ("Chota Chattri"): Handles daily group broadcasts, social posting workflows, general watchlists, and manual application assists.
  • Agent `jaskirat`: Orchestrates 9 highly focused industry category crawls (Services, AI, Cyber, Web3/Fintech) keeping data separated in local databases.
  • Personalization Digest Engine: Processes candidate preferences privately in DM, mapping aggregated daily scans against individual profiles.

Key learnings & Insights

Valuable discoveries from building a collaborative multi-agent pipeline:

  • Task Segmentation (Sector Slices) solves context bottlenecks: Splitting scans into discrete categories (Services, Web3, Cyber) reduces context lengths, prevents token usage spikes, and prevents multi-company crawl timeouts.
  • Incremental Deduplication via Checkpoints: Daily crawlers generate massive redundant data. Writing daily Markdown memory logs (`checkpoint_*.md`) allows the agent to check and exclude duplicates prior to LLM evaluation.
  • Multi-Agent Isolation for Privacy: Storing candidates' preferences privately in `job_users.json` and generating individual DMs prevents profile leakages in public group channels.
  • X / Twitter Social Post Loop: Pulling the newest Telegram notes, translating, rewriting under 280 chars, and auto-publishing drafts is an excellent loop to build developer public presence.
